Detailed Description

Base main window class, use instead of QMainWindow.

Provide basic functionality including: menus, toolbars, status bar, handling events (including settings change), interface to read and write working file.

Using: 1) Inherit from UtilityMainWindow. 2) Initialize UI in constructor, by Qt documentation. Add you widget for example. 3) Override getExtensions(), getFileReadMode(), getFileWriteMode() to specified file types and read-write modes. 4) Override loadFile(fileName) and saveFile(filename) implementing read and write file from disc. See example for basic variant. 5) Delegate cut(), copy(), paste() to your widget.

Instantiate and use as normal: MainWindow mainWin; <- heir of UtilityMainWindow. mainWin.show();
