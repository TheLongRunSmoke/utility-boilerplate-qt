Detailed Description

Application settings.

Things like window state, recent files, etc. already included in Settings class.

Add parameters that you need using addUserSetting(<some settings item>) in createUserSettings(), than it's can be accessed by Qt value(key).

Note, start add userSectionTag() to begin of key.
